服务条款

使用 FaroAI 服务前，请仔细阅读以下条款。

一、服务范围

FaroAI（以下简称"我们"或"平台"）向用户提供基于人工智能技术的智能助手服务，包括但不限于以下功能：

  • AI 对话服务：基于大型语言模型的文本、图像、语音等多模态对话交互
  • 知识库管理：文档上传、存储、检索和基于文档的智能问答
  • 自动化工作流：可视化搭建和执行自动化任务流程
  • API 服务：提供 REST API 接口，供第三方系统集成
  • 团队协作：多人共享知识库、协同编辑和权限管理

具体可用的功能取决于你所订阅的方案版本。免费功能可能会调整或终止，我们会提前 30 天通知。

二、用户义务

2.1 账户安全

你需要妥善保管自己的账户信息和 API Key，不得将账户共享给他人使用。因账户信息泄露导致的一切后果由用户自行承担。

2.2 合法使用

使用 FaroAI 服务时，你承诺遵守以下规定：

  • 遵守所有适用的法律法规
  • 不利用服务从事任何违法、侵权或不当行为
  • 不利用 AI 生成虚假信息、欺诈内容或恶意软件
  • 不生成、传播任何色情、暴力、恐怖主义或其他违禁内容
  • 不侵犯他人的知识产权、隐私权或其他合法权益
  • 不超过所订阅方案的速率限制和资源配额

2.3 内容责任

你对其通过 FaroAI 生成的所有内容（包括 AI 输出结果）负责。AI 生成的内容可能存在不准确或不适当的情况，用户需要自行判断和审核后再使用。

五、终止条款

5.1 用户终止

你可以随时终止使用 FaroAI 服务。终止后，你的账户数据将在 30 天后被永久删除。如果你需要导出数据，请在终止前通过设置面板导出。

5.2 平台终止

在以下情况下，我们有权终止你的账户和服务：

  • 违反本服务条款中的任何规定
  • 滥用服务资源（如超出速率限制、恶意调用 API）
  • 使用服务从事违法活动
  • 长时间未活跃（超过 12 个月）

我们会在终止前通过邮件通知你（严重违规除外）。终止后未消费的预付费余额将按剩余天数退还。

5.3 服务终止

如果我们决定停止运营 FaroAI 服务的全部或部分内容，会提前至少 60 天通过邮件和网站公告通知用户，并提供数据导出工具和退款方案。
