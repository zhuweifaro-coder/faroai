贡献指南

如何参与贡献

我们欢迎任何形式的贡献，无论大小。

💻 代码贡献

  1. Fork 仓库并 clone 到本地
  2. 创建功能分支 feature/xxx
  3. 遵循代码规范，编写测试
  4. 提交 Pull Request
  5. 等待 Maintainer Review 并合并

📝 文档贡献

  1. 完善 API 文档和示例代码
  2. 翻译文档到更多语言
  3. 编写教程和使用指南
  4. 修正拼写和格式错误
  5. 提交文档 PR 或直接在 GitHub 编辑

🔌 插件开发

  1. 阅读插件开发文档
  2. 使用插件模板快速创建
  3. 本地测试和调试
  4. 提交到插件市场审核
  5. 审核通过后面向社区发布
